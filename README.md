## 프로젝트 목표
- 데이터 엔지니어로 직무전환을 위한 데이터 파이프라인 설계
- 데이터 품질관리, 개인정보 보호 적용 초점

## 경력과 연관성
- 공공기관 개인정보보호(3년), 제조업 품질관리(5년) **프로세스 관리**와 **규제 준수** 경험 

## 사용 기술
- Python : 데이터 처리 및 분석
- Apache Airflow : 데이터 파이프라인 자동화
- SQL : 데이터베이스 관리
- Cryptography : 데이터 암호화
- AWS S3 : 데이터 저장소

## 진행 단계
- 데이터 수집 : 개인정보가 포함된 데이터를 탐지하고, 이를 안전하게 저장하는 방법 설계
- 데이터 저장 : 암호화 및 보안이 강화된 데이터 저장소 설정
- 데이터 분석 : 분석을 위해 개인정보를 안전하게 비식별화
- 데이터 보존 및 삭제 : 데이터 보유 기간을 관리하고, 주기적으로 삭제하는 시스템 구현
